# Flipkart Sales Dashboard (Power BI Project)

## Project Overview
This project presents an interactive Power BI dashboard that analyzes Flipkart sales data. 
The dashboard provides insights into sales performance, product categories, regional sales, 
and order trends using data visualization techniques.

The goal of this project is to transform raw sales data into meaningful business insights 
using Power BI.

---

## Tools & Technologies Used

- Microsoft Power BI
- Data Visualization
- Data Modeling
- Data Cleaning
- Excel Dataset

---

## Dataset Description

The dataset contains Flipkart order information including:

- Order ID
- Order Date
- Sales Channel
- Fulfillment
- Product Category
- Quantity Ordered
- Sales Amount
- Ship City
- Ship State
- Ship Country
- Courier Status

---

## Dashboard Features

The Power BI dashboard includes the following visualizations:

- Total Sales Overview
- Sales by Product Category
- Sales by State / Region
- Order Status Distribution
- Sales Trends Over Time
- Quantity Sold Analysis
- B2B vs B2C Orders

---

## Key Insights

- Certain product categories generate the highest revenue.
- Sales trends show seasonal fluctuations.
- Some states contribute significantly more to total sales.
- Majority of orders belong to B2C customers.

## How to Use

1. Download the `.pbix` file.
2. Open it in **Microsoft Power BI Desktop**.
3. Explore the interactive dashboard.
4. Use filters and slicers to analyze different sales insights.
